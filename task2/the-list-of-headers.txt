age: кількість секунд з моменту зміни ресурсу.
cache-control: 	основні значення для керування кешуванням.
content-encoding: спосіб кодування вмісту даних при передачі.
content-type:	формат і спосіб представлення даних.
date: дата створення відгуку.
etag: тег версії даних, що використовується при кешуванні.
last-modified: дата останньої зміни даних.
server: список назв і версій веб-серверу і його компонентів з коментарями. 
status: показує, чи був успішно завершений певний HTTP-запит.
strict-transport-security: механізм, який примусово активує захисне з'єднання через протокол HTTPS
vary: список полів взяті із запиту, що описують ресурс.
via: 	список версій протоколу, назв і версій проксі-серверів, через котрих пройшло повідомлення.
:method: метод позначення запиту для бажаної дії над певним ресурсом
:path: шлях запиту для бажаної дії над певним ресурсом
accept: список допустимих форматів ресурсу.
accept-encoding:	перелік підтримуваних способів кодування вмісту даних при передачі.
accept-language: список підтримуваних мов.
cookie: збереження даних про клієнта браузером
if-modified-since: дата при зміні даних з вказаного моменту.
if-none-match: список тегів версій даних
user-agent: список назв,версій та компонентів з коментарями клієнта.
upgrade-insecure-requests - говорить серверу про те,що браузер хоче отримувати сайт в захищеному режимі
Connection: Дані про проведення з'єднання.
Content-Language: Один або кілька мов вмісту даних.
Expires: Дата закінчення терміну актуальності даних.	
Pragma: особливі опції виконання дії.
Set-Cookie:запит до браузера про збереження кукі
Transfer-Encoding: Список способів кодування, котрі були використані до повідомлення для передачі.
X-Frame-Options:використовується для додаткової безпеки, яка надається лише у випадку, якщо користувач, який отримує доступ до документа, використовує веб-переглядач, що підтримує файл X-Frame-Options 
Host: Доменне ім'я і порт  ресурсу
content-length: розмір вмісту даних в байтах.
alt-svc: повідомляє клієнту про наявність альтернативного сервісу
x-xss-protection: попередження атаки
